Make sure VcXsrv or XQuartz is running each time you run graphics

(***************Windows****************)
install VcXsrv: https://techcommunity.microsoft.com/t5/windows-dev-appconsult/running-wsl-gui-apps-on-windows-10/ba-p/1493242

enter in ubuntu command line:
$ sudo apt install ocaml-tools libsdl-ocaml-dev
$ opam install opam-depext
$ opam depext conf-pkg-config
$ opam install graphics imagelib camlimages
$ sudo apt install -y xserver-xorg-core
$ cd ~
$ ls -a (look for .bashrc or .bashprofile)
$ echo 'export DISPLAY=:0.0' >> .bashrc
$ echo $DISPLAY   
(above command should output localhost:0)

(*****************Mac*****************)
download & launch XQuartz

enter in the terminal:
$ sudo port install libsdl libsdl_mixer
$ opam install -y ounit yojson graphics camlimages
$ opam install ocamlsdl
